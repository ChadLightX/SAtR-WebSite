Site Sun After the Reign:



Pourquoi faire un site:

1- Mettre en avant l'association Sun After The Reign

2- Avoir la possibilité de communiquer plus facilement entre l'association et ses particuliers

3- Poster une carte de visite en ligne en valorisant ses activités et ses services



Idée du site (brainstorming du site web):

\-Page d'accueil

&#x09;-Présentation globale du site

&#x09;-Lien en bas de page pour:

&#x09;	-La charte Graphique

&#x09;	-La FAQ

&#x09;	-Les mentions légales

&#x09;	-Les moyens de contacts de l'association

&#x09;	-Les différents liens des réseaux sociaux de l'association

\-Page Qui Sommes-nous

\-Page Nous contacter

\-Page Prochains évènements

\-Page de Ticketing (envoyeur)

\-3 différents comptes:

&#x09;-Compte utilisateur

&#x09;-Compte Administrateur

&#x09;	-Gestion des comptes

&#x09;	-Résolution des problèmes du site

&#x09;	-Page de ticketing (receveur) 

&#x09;	-Mise à jour du site web

&#x09;-Compte membre SAtR

&#x09;	-Ajout des prochains évènements

&#x09;	-Mise à jour du classement

\-Page Classement par saison (API Challonge + Google Tab)

\-Page Carte de France avec toutes les associations/prochain tournois

\-Page Agenda de combo

\-Page Anciens évènements

\-Page de merchandising pour les ventes de produits

\-Mode Portable du site web



Le tout doit respecté les closes de sécurité établi par la CNIL et doit avoir un nom de domaine valide.



Aspect Technique du site

Si gestion des utilisateurs alors création d'une base de donnée solide.

Si gestion d'évènements alors création d'une base de donnée solide.

API Challonge + Google Tab pour les classements par saison sur Paris ou sur Marseille 

API Discord pour la redirection de lien vers ce dernier ou le besoin en information de celui ci

Frontend en CSS

Backend via un Framework (sans doute React.js, Riot.js voir en Type Script)

Index en html pour avoir la base de la page d'accueil du site 

Le Backend doit pouvoir faire le lien entre la base de données et le site

Le Frontend doit pouvoir afficher le contenu des requêtes du site
